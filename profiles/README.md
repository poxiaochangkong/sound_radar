# Profiles

This directory holds **optional** per-game calibration files generated by
`tools/calibrate.py`. Each file overrides the defaults in `config.yaml` for
a specific game or environment.

## Important

The core application **never depends on any profile**. If this directory is
empty, the app runs with the game-agnostic defaults in `config.yaml`.

## Usage

    python tools/calibrate.py --game csgo
    python src/main.py --profile profiles/csgo.yaml
