# 07 — 安全与合规免责声明

## 1. 项目性质

本项目是一个**音频分析原型与学习项目**,用于研究:

- Windows WASAPI loopback 多声道捕获
- 多声道能量分析与方位估计(VBAP)
- 实时音频事件检测(spectral flux)
- PyQt6 实时可视化

它的目的是**技术验证与教学**,不是商业产品,也不是竞技作弊工具。

## 2. 合规风险

⚠️ **重要警告**:在某些多人竞技游戏中,使用此类工具可能:

- 违反游戏的服务条款(Terms of Service)
- 触发反作弊系统的检测(即使本工具不读取游戏内存)
- 导致账号被封禁

具体是否违规取决于每款游戏的 EULA 和反作弊策略。**使用者有责任自行核实并承担后果。**

## 3. 适用场景

本项目**适合**用于:

- 音频处理与 DSP 学习
- 多声道音频分析实验
- PyQt / Qt Painter 可视化练习
- Windows WASAPI loopback 研究
- 单机游戏 / 训练场 / 自定义对局的音频调试

本项目**不建议**用于:

- 官方排位赛
- 有反作弊系统保护的多人对局
- 任何可能影响他人游戏体验的竞技场景

## 4. 技术限制

即使不考虑合规问题,本工具也有固有限制(见 `05_algorithm.md` 第 8 节):

- 无法识别上下方位
- 强依赖真实多声道输出链路
- 单声源假设(多声源会偏向能量大者)
- 正前方分辨率受 C 声道忽略影响

**它是一个"声音方向提示器",不是精确的敌人定位系统。**
